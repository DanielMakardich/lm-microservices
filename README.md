# lm-dashboard
The following repo contains all of the microservices developed for the Logimethods Dashboard.  

The following services are in development:
 
* Experience Layer:
  * lm-dashboard-ui

* Domain Layer:
  * lm-timesheet-domain

* System Layer:
  * lm-employee-sys
  * lm-timesheet-sys
  * lm-project-sys
  * lm-customer-sys
